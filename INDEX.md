## 🗂️ Flake Structure

### Core Files
- **flake.nix** - Main entry point (defines nixOS and home-manager configurations)
- **flake.lock** - Lock file (generated by `nix flake update`)

### System Configuration (`hosts/cachyos-to-nix/`)
- **default.nix** - Host imports and system-level settings
- **hardware.nix** - Hardware config (UUIDs, filesystems, boot) - **YOU MUST UPDATE THIS**
- **networking.nix** - Network settings, hostname, firewall

### NixOS Modules (`modules/`)

#### Desktop (`modules/desktop/`)
- **kde.nix** - KDE Plasma 6 complete setup
- **gnome.nix** - GNOME desktop complete setup
- *Both run simultaneously, switch at login*

#### Hardware (`modules/hardware/`)
- **nvidia.nix** - NVIDIA RTX 2070 SUPER drivers and CUDA

#### System (`modules/system/`)
- **boot.nix** - Bootloader (GRUB), kernel configuration
- **localization.nix** - Timezone, locale, keyboard, fonts
- **services.nix** - PipeWire, Bluetooth, CUPS, Docker, etc.

#### Development (`modules/development/`)
- **dev-tools.nix** - 317+ dev packages (compilers, IDEs, tools)

#### Gaming (`modules/gaming/`)
- **gaming.nix** - Steam, Lutris, Proton, Wine, emulators

#### Shell (`modules/shell/`)
- **shell-config.nix** - Zsh, bash, shell utilities, modern CLI tools

### Home-Manager Configuration (`home-manager/`)

User-level configuration managed by Home-Manager.

#### Main Config
- **default.nix** - Home-manager imports and user packages

#### Modules (`home-manager/modules/`)

**Shell** (`shell/`)
- **default.nix** - Zsh with Powerlevel10k, git, direnv, fzf

**Desktop** (`desktop/`)
- **default.nix** - XDG dirs, GTK/Qt themes, MIME types
- **kde.nix** - KDE-specific home-manager config
- **gnome.nix** - GNOME-specific home-manager config (dconf settings)

**Development** (`development/`)
- **default.nix** - IDEs, language servers, containers, databases

**Media** (`media/`)
- **default.nix** - Creative software (GIMP, Blender, video editors)

**Gaming** (`gaming/`)
- **default.nix** - Gaming configs (Steam, Lutris, MangoHUD)

## 🚀 Quick Start

### 1. Read First
→ Start with [SETUP.md](SETUP.md)

### 2. Critical Step
Update `hosts/cachyos-to-nix/hardware.nix` with your actual UUIDs and filesystem config.

### 3. Build
```bash
cd /etc/nixos
sudo nixos-rebuild switch --flake .#cachyos-to-nix
```

### 4. Reboot & Enjoy
Switch between KDE and GNOME at the SDDM login screen!

## 🔧 Customization

### Add a Package
Edit appropriate file and rebuild:
```bash
# System package
sudo nano /etc/nixos/home-manager/default.nix
# Add to home.packages

home-manager switch --flake /etc/nixos#radu
```

### Disable a Feature
Comment imports in `hosts/cachyos-to-nix/default.nix`:
```nix
# ../../modules/gaming/gaming.nix  # Disable gaming
```

Rebuild:
```bash
sudo nixos-rebuild switch --flake .#cachyos-to-nix
```

### Switch Desktop
At SDDM login: Click "Session" → Select "Plasmawayland" or "GNOME" → Log in

### Configure Shell
Edit `home-manager/modules/shell/default.nix` and rebuild.

## 🔑 Key Configuration Points

**Must Update Before Install:**
- `hosts/cachyos-to-nix/hardware.nix` - Your disk UUIDs

**Should Update After Install:**
- `home-manager/modules/shell/default.nix` - Git email
- `hosts/cachyos-to-nix/networking.nix` - Hostname (optional)

**Optional Customizations:**
- Add/remove packages in any `default.nix`
- Enable/disable modules in `hosts/cachyos-to-nix/default.nix`
- Customize themes, keybinds, etc. in individual files
